# Abdullah Taufiq — Portfolio

A single-page, animated portfolio site built from Abdullah's resume.

## What's inside
- `index.html` — the whole site (HTML/CSS/vanilla JS, no build step)
- Interactive canvas "knowledge graph" hero, scroll-reveal sections, animated timeline

## Run locally
Just open `index.html` in a browser, or serve it:

```
python3 -m http.server 8000
```

## Deploy on GitHub Pages
1. Create a repo (e.g. `abdullah-portfolio`) and push this file to it.
2. In the repo settings, enable **Pages** → source: `main` branch, `/ (root)`.
3. Your site will be live at `https://<username>.github.io/<repo>/`.
